# WhatsApp-Auto-SMS-Sender-tool
A Python-based WhatsApp automation tool that helps automate messaging tasks such as sending messages, managing contacts, and interacting with WhatsApp Web. Built using Selenium, this project is designed for learning automation, improving productivity, and exploring real-world browser automation with Python.

📲 WhatsApp Python Automation
🚀 Overview

This project is a Python-based WhatsApp automation tool built using Selenium. It automates tasks on WhatsApp Web such as sending messages and managing contacts, helping users save time and learn browser automation.

⚙️ Features
📤 Send automated WhatsApp messages
👥 Extract and manage contacts
🌐 Works with WhatsApp Web
🤖 Beginner-friendly automation scripts
📁 Easy to customize and extend
🛠️ Tech Stack
Python 🐍
Selenium 🌐
Chrome WebDriver
📦 Installation
1. Clone the repository
git clone https://github.com/saboormalikawan512-cmyk/whatsapp-automation.git
cd whatsapp-automation
2. Install dependencies
pip install -r requirements.txt
▶️ Usage

Run the main script:

python ProGUI.py

Then:

Open WhatsApp Web
Scan QR code
Choose automation option (send messages, etc.)
